# Future_in_Dsca_T3
Marketing funnel &amp; Conversion performance analysis

In this task we look at bank marketing campaign dataset to perform a marketing funnel and conversion performance analysis.
We analyze the dataset to see how customers progress from initial contact to successful subsciption. We also evaluate conversion rates, channel effectiveness and drop-off points.
Firstly i used excel to view the dataset and analyzed with visual studio code(python), i therefore created a pdf report of the analysis and provided recommendations to improve marketing return on investment significantly.
